# DigiTools 🛠️

A modern digital tools marketplace where users can browse and purchase premium digital products including AI writing tools, design templates, stock assets, and more.

## 🚀 Live Site
[Live Demo](https://digitools-tan.vercel.app/)

## 🛠️ Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- Vite

## ✨ Features
- 🛒 **Cart System** — Add, remove products and proceed to checkout with real-time cart count in navbar
- 🔔 **Toast Notifications** — Get instant feedback when adding to cart, removing items or checking out using React-Toastify
- 🔄 **Product & Cart Toggle** — Seamlessly switch between browsing products and managing your cart in one section
